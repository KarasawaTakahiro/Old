nicoDL_dev
==========
ニコニコ動画からURLをもとに動画を保存します。  
動画説明文やサムネイルなど付属情報も保存するバージョンがあります。  


##バージョン遷移

1. nicoDL_0,1
2. nicoDL
3. nicoDL_v0,90
4. nicodl3

## 補足
.ndlファイルを見るにはnicoDL_LibraryViewerが必要

## 環境
* windowsXP
* python 2.6
* wxPython 2.7

##実行
windowsXPで動作はしますが、フォルダ指定がハードコードのため、動作はしません。  
nicoDL.py もしくは nicoDL_main.pyを実行します。
